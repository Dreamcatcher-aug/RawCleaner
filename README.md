# RawCleaner 使用手册

</div>
**软件版本**：v1.0 beta
**适用系统**：Windows
**功能**：自动识别无对应 JPG 的 RAW 废片，安全清理至回收站

### 一、界面说明
1. **左侧拖拽框**：用于拖入需要扫描的照片文件夹
2. **右侧状态栏**：显示扫描进度、找到的废片信息
3. **底部按钮**：开始对比 → 确认处理 → 重置

### 二、使用步骤（必看）
#### 1. 导入文件夹
将你的照片文件夹**直接拖拽到左侧框内**
支持同时拖入多个文件夹

#### 2. 开始对比
点击 **【开始对比】**
软件自动遍历所有文件夹，识别：
- 无对应 JPG 的 RAW 格式废片
- 实时显示在右侧状态栏

#### 3. 确认处理
1. 点击 **【确认处理】**
2. 弹出**二次确认弹窗**
3. 确认后，废片自动移入**回收站**（可恢复）

#### 4. 重置（必须操作）
清理完成后，**必须点击【重置】**
才能进行下一次扫描处理

### 三、安全说明
✅ 所有删除文件仅进入回收站，不会永久删除
✅ 二次确认弹窗，防止误操作
✅ 不修改、不删除任何有对应 JPG 的 RAW 文件

### 四、注意事项
1. 每次操作完成**必须点重置**，否则无法继续使用
2. 支持所有主流相机 RAW 格式
3. 路径不要包含特殊符号，避免扫描失败
4. 若无法拖拽，请用**管理员身份运行**软件
